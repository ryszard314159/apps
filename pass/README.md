# Password generator

Use command in `customize.sh` to create
custom `pass.js` and `pass.html` files.


# Usage

## Commad line

`./pass.js -h`

See https://nodejs.org

Build self-contained package:
```
$ npm install -g pkg
$ pkg pass.js -o pass
$ ./pass -h
```

## Browser

Open `pass.html` in your browser
